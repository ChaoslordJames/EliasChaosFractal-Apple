## Installation
```bash
# Clone the chaos
git clone https://github.com/ChaoslordJames/EliasChaosFractal-Apple.git
cd EliasChaosFractal-Apple

# Fetch dependencies
swift package update

# Initialize IPFS (optional, for full swarm)
ipfs init

# Start Redis
redis-server --daemonize yes
```

## Run the Swarm
```bash
# Launch the P2P node
swift run P2PNode --peer-id QmChaosLordJames
```

## Run the GUI
